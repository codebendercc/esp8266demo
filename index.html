ESP826 Demo

Getting started with the ESP8266 in seconds!

Hey guys, and welcome to our little ESP8266 Getting Started page! This is provided by codebender as a demo of the things we're working on for ESP8266.

Please note that this page is currently available for the Adafruit HUZZAH ESP8266 and SparkFun Thing Dev boards.

This page will guide you through using your ESP8266 board for the first time, help you set up your computer to program your ESP8266 board, and it will also show you some of the capabilities of the boards.

We're using codebender, which not only features code-sharing, but also allows anyone to upload and debug Arduino sketches from within a web browser. Plus, it works on lower-tech machines, like Chromebooks!

Before starting, you need to install the right Browser Plugin. Click here for Chrome or here for Firefox.

Hello World

Another one of the most simple programs you can write is the Hello World program. This sends data from your ChipKIT board to your computer

The code itself is pretty easy to understand. Go ahead, click Flash to load this code to your board.

After uploading, go to the embedded serial monitor below, set the baud rate to 115200 bps, and click Connect. You should start getting data from your ChipKIT board.
